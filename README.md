# JavaScript Mini Projects

A collection of beginner-friendly JavaScript projects built while learning core JavaScript concepts and problem-solving techniques.

These projects focus on strengthening fundamental JavaScript skills such as:

* Variables and Data Types
* Functions and Arrow Functions
* Arrays and Objects
* Loops
* Array Methods (`forEach`, `find`, `findIndex`, `filter`, `reduce`)
* Template Literals
* Problem Solving
* Clean Code Practices

---

## Projects

### 1. Student Management System

A console-based application to manage student records.

#### Features

* Add students
* Display all students
* Find student by ID
* Calculate average marks
* Find topper
* Remove student by ID
* Count total students
* Filter students above a specific mark
* Calculate student grades

#### Concepts Used

* Arrays
* Objects
* Arrow Functions
* `forEach()`
* `find()`
* `findIndex()`
* `filter()`
* `reduce()`
* Conditional Statements
